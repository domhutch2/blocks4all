# Regional Anaesthesia Notes

A searchable reference for common surgical procedures and their recommended regional anaesthetic techniques. **#Blocks4ALL**

## How to use this guide

Use the **search bar** at the top to find a procedure, block, or drug by name. Or browse by body region using the **sidebar**.

Don't know the procedure name? Use the **[Find by Body Part](./find-by-body-part.md)** page to look up by presentation.

## Quick Reference

Essential pages you'll need on shift:

- **[LA Dosing & Calculations](./quick-reference/la-dosing.md)** — max doses, concentration tables, adjuvants
- **[LA Toxicity & Lipid Rescue](./quick-reference/la-toxicity.md)** — recognition, Intralipid protocol, cardiac arrest management
- **[Standard Infusion Recipes](./quick-reference/standard-infusions.md)** — epidural, PVB, fascia iliaca, sciatic catheters
- **[Antibiotic Re-dosing](./quick-reference/antibiotic-redosing.md)** — intraoperative re-dosing intervals by agent
- **[Contacts](./quick-reference/contacts.md)** — pain team, RA on-call, useful links

## Each procedure page includes

- **Plan A** — the recommended regional technique with dosing
- **Plan B / alternatives** — for when the primary block isn't feasible
- **Key anatomy** — relevant landmarks and sonoanatomy
- **Considerations** — contraindications, practical tips, patient factors
- **References** — PROSPECT, NICE, primary evidence
