# Anticoagulation & Blocks

Timing of nerve blocks in anticoagulated patients, based on ESRA 2022 guidelines and local guidance.

:::warning Key Principle
The classification of a block as **deep** or **superficial** determines whether anticoagulation timing rules apply. Superficial blocks may generally be performed without cessation of anticoagulants. Deep blocks require consideration of timing from last dose.
:::

## Deep vs superficial block classification

Based on ESRA 2022 guidelines, reformatted from local guideline Appendix B.

| Deep nerve blocks | Superficial nerve blocks |
|---|---|
| Stellate ganglion, deep cervical plexus, cervical paravertebral | Occipital, peribulbar, sub-Tenon's, superficial cervical plexus |
| Infraclavicular | Interscalene, supraclavicular, axillary, suprascapular, forearm and wrist blocks |
| Epidural, thoracic paravertebral | ESP, serratus plane, parasternal intercostal, interpectoral and pecto-serratus (PECS) |
| | Ilioinguinal, TAP, rectus sheath, pudendal |
| Quadratus lumborum, lumbar plexus, psoas compartment, lumbar paravertebral, sacral plexus, proximal sciatic, spinal, epidural | Femoral, adductor canal, sciatic (subgluteal or popliteal level only), fascia iliaca, LCNT, ankle blocks |

## Timing of deep nerve blocks with anticoagulants

Reproduced and rephrased from ESRA 2022 guidelines (local guideline Appendix C).

| Drug / dose | Time from last dose to block | Target lab value | Time from block to next dose |
|---|---|---|---|
| **Warfarin** | Until INR normal (usually 5 days) | Normal range for patient | — |
| **DOAC** (rivaroxaban, edoxaban, apixaban) — CrCl > 30 | 48h | No testing required | 24h |
| **DOAC** — CrCl < 30 | 72h | No testing required | 24h |
| **LMWH prophylactic** (e.g. tinzaparin ≤ 50 units/kg/day) | 12h (24h if CrCl < 30) | No testing required | — |
| **LMWH therapeutic** | 24h (48h if CrCl < 30) | Anti-Xa ≤ 0.1 IU/mL | — |
| **UFH — lower dose** (≤ 100 IU/kg/day IV) | 4h | No testing required | — |
| **UFH — higher dose** | Until target lab value (~6h if IV) | aPTT or anti-Xa in normal range | — |
| **Fondaparinux ≤ 2.5 mg/day** | 36h (72h if CrCl < 50) | No testing required | — |
| **Fondaparinux higher dose** | Until target lab value (~4 days) | Calibrated anti-Xa ≤ 0.1 IU/mL | — |
| **Aspirin** | None — can be performed immediately | No testing | Aspirin can resume at any point |
| **P2Y12 inhibitor** | Ticagrelor: 5 days; Clopidogrel: 5–7 days; Prasugrel: 2 days (or until lab value) | None routinely, consider platelet function testing | Clopidogrel 75 mg: 0h; Clopidogrel 300 mg: 2 days; Prasugrel/ticagrelor: 1 day |
| **Aspirin + antiplatelet/anticoagulant** | As per the antiplatelet/anticoagulant | As per antiplatelet/anticoagulant | Aspirin can resume at any point |

:::info Superficial blocks
Superficial nerve blocks may all be performed **without cessation** of anticoagulant or antithrombotic drugs, provided any relevant level (INR, anti-Xa) is within or below the therapeutic range, and there is no clinical suspicion of overdose or drug accumulation.
:::

## Key points

- The ESRA guidelines separate blocks into deep and superficial according to risk of provoking non-compressible or catastrophic bleeding
- For **deep blocks** in anticoagulated patients: consider timing carefully, document rationale, and discuss with supervising consultant if performed by a trainee
- Performing a block that contravenes ESRA timing (e.g. for acute trauma) must be a documented risk-benefit decision in the patient's notes and consent form
- **Therapeutic LMWH should not be given when a nerve or epidural catheter is in situ**
- Trainees performing blocks on anticoagulated patients must have appropriate supervision
